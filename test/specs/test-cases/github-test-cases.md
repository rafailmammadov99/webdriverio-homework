# GitHub Automation Homework

## Test Case 1 – GitHub Sign Up

**Steps**
1. Open https://github.com
2. Click "Sign up"
3. Verify text "Create your account"
4. Enter email
5. Click Continue
6. Enter password
7. Click Continue
8. Enter username
9. Click Continue
10. Choose email preferences
11. Click Continue

**Expected Result**
User proceeds through GitHub registration steps.
